12.1 전문 검색

전문 검색 : 용량이 큰 문서를 단어 수준으로 잘게 쪼개어 문서 검색을 하게 해주는 기능

12.1.1 전문 검색 인덱스의 생성과 검색

- 형태소 분석 : 문장의 공백과 같은 띄어쓰기 단위로 단어를 분리, 각 단어의 조사를 제거해서 명사 또는 어근을 찾아서 인덱싱하는 알고리즘
- n-gram 파서 : 공백과 같은 띄어쓰기 단위로 단어를 분리, 단어를 주어진 길이로 쪼개서 인덱싱하는 알고리즘
